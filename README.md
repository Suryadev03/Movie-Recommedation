# 🎬 Movie Recommendation App

A Streamlit‑based application that recommends movies using AI/ML techniques.  
Built with Python, Streamlit, and machine learning libraries.

WebLink : https://movie-recommedation-bzappvkhqisprpbaqnst6bz.streamlit.app/

---

## 🚀 Features
- Upload or input movie preferences
- Generate personalized recommendations
- Interactive Streamlit UI
- Easy deployment on Streamlit Cloud

---

## 🛠️ Tech Stack
- **Python 3.10+**
- **Streamlit** for UI
- **Pandas / NumPy** for data handling
- **Scikit‑learn / Surprise** for recommendation algorithms
- **Google Gemini API** (optional, for AI‑assisted features)
